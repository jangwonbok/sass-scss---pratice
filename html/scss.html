1. Sass / Scss의 차이
Sass가 처음 나왔을때 문법은 CSS와 달랐는데 이 문법이 SASS버전 3이상 부터 SCSS문법으로 변경 되었다.

- Sass 문법 구조

초기 SASS의 문법은 CSS 문법에서 사용되는 {},;등을 사용하지 않는다. $bgc: gold $colr: #333 body{ background-color:$bgc color:$colr }

- Scss 문법 구조

현재 우리가 사용하고 있는 Sass의 문법. CSS3업데이트 버전으로 CSS Syntax와 비슷하면서 유연한 문법 등을 제공한다. $bgc: gold; $colr: #333; body{ background-color:$bgc; color:$colr; }
2. 기본 문법 - 변수(Variables)

① 일반 변수

반복적으로 많이 사용되는 부분을 변수로 지정해 놓고 변수값만 변경하여 사용한다. ex) $변수명 : 속성값;

Scss

$bgc: #333; $point-color: #ddd; .box{ background-color:$bgc; color:$point-color; }
▼

CSS

.box{ background-color:#333; color:#ddd; }

② global 변수

global 변수: 모든 영역에서 사용 가능

Scss

$width: 5px; .box{ $height: 5px !global; // 규칙 내부에 작성했지만 전역 변수 사용 width:$width; height:$height; } .box2{ height:$height; // !global 플래그로 작성한 전역 변수 사용 }
▼

CSS

.box{ width:5px; height:5px; } .box2{ height:5px; }

③ local 변수

해당 소괄호 안에서만 사용가능한 변수

Scss

.box{ $padding:5px; padding:$padding; .box-cont{ padding:$padding; // 하위 규칙에도 사용가능 } } .box2{ padding:$padding; //지역 변수 범위를 벗어남 해당 지역에 변수가 없으므로 에러 발생.. }
▼

CSS

.box{ padding:5px; } .box .box-cont{ padding:5px; } .box2{ error; }
3. Nesting(네스팅) 중첩
중괄호 안에 하위 요소에 대한 스타일을 지정하는 형태로 CSS 선택자를 중첩 할 수 있다.

Scss

.nesting{ background-color:gold; ul{ margin-top:10px; padding-top:10px; } li{ margin-top:10px; a{ display:block; color:red; } } }
▼

CSS

.nesting{ background-color:gold; } .nesting ul{ margin-top:10px; padding-top:10px; } .nesting li{ margin-top:10px; } .nesting li a{ display:block; color:red; }
4. Ampersand (상위 선택자 참조)
중첩 안에서 & 키워드는 상위(부모) 선택자를 참조하여 치환합니다.

Scss

.box{ position:absolute; &.active{ color:red; } } .list{ li{ &:last-child{ margin-right:0; } } }
▼

CSS

.box{ position:absolute; } .box.active{ color:red; } .list li:last-child{ margin-right:0; }
5. @at-root (중첩 벗어나기)
중첩에서 벗어나고 싶을 때 @at-root 키워드를 사용합니다. 중첩 안에서 생성하되 중첩 밖에서 사용해야 유용

Scss

.list{ $w : 100px; $h : 50px; li{ width:$w; height:$h; } @at-root .box{ width:$w; height:$h; } }
▼

CSS

.box{ position:absolute; } .box.active{ color:red; } .list li:last-child{ margin-right:0; }
6. 중첩된 속성
font-, margin-, padding- 등과 같이 동일한 네임 스페이스를 가지는 속성들을 다음과 같이 사용할 수 있다.

Scss

.box{ font{ weight:bold; size:10px; } margin{ left:10px; right:10px; } padding{ left:10px; right:10px; } }
▼

CSS

.box{ font-weight:bold; font-size:10px; margin-left:10px; margin-right:10px; padding-left:10px; padding-right:10px; }
7. 가져오기 (Import)
@import로 외부에서 가져온 Sass 파일은 모두 단일 CSS 출력 파일로 병합됩니다.
또한, 가져온 파일에 정의된 모든 변수 또는 Mixins 등을 주 파일에서 사용할 수 있습니다.

Scss

@import "path/filename";
8. @mixin, @include

① @mixin

Mixin는 스타일 시트 전체에서 재사용 할 CSS 선언 그룹 을 정의하는 기능 @mixin 믹스인이름{ 스타일; }

② @include

선언된 Mixin을 사용(포함)하기 위해서 선언하는 기능

Scss

@mixin btnStyle{ display:inline-block; padding:10px 10px; font: { size:10px; wieght:bold; } color:red; } .box{ @include btnStyle; } .box2{ @include btnStyle; }
▼

CSS

.box{ display:inline-block; padding:10px 10px; font-size:10px; font-weight:bold; color:red; } .box2{ display:inline-block; padding:10px 10px; font-size:10px; font-weight:bold; color:red; }
9. @mixin 인수값

① @mixin 인수

Mixin은 함수 처럼 인수값을 가질 수 있다. 하나의 Mixin으로 다양한 결과를 만들 수 있다. @mixin 믹스인이름($매개변수){ 스타일; } @include 믹스인이름(인수);

Scss

$color: #222; @mixin border-line($width, $color){ border:$width solid $color; } .box1{ @include border-line(5px, $color); }
▼

CSS

.box1{ border:5px solid #222; }

② @mixin 인수 기본값 설정

인수값은 기본값을 가질 수 있습니다.
@include 포함 단계에서 별도의 인수가 전달되지 않으면 기본값이 사용됩니다. @mixin 믹스인이름($매개변수: 기본값){ 스타일; }

Scss

@mixin border-line($width: 1px, $color:blue){ border:$width solid $color; } .box1{ @include border-line; } .box2{ @include border-line(4px); }
▼

CSS

.box1{ border:1px solid blue; } .box2{ border:4px solid blue; }

③ 키워드 인수

mixin에 전달할 인수를 입력할 때 명시적으로 키워드(변수)를 입력하여 작성할 수 있습니다.
별도의 인수 입력 순서를 필요로 하지 않아 편리하게 작성할 수 있습니다.
단, 작성하지 않은 인수가 적용될 수 있도록 기본값을 설정해 주는 것이 좋습니다. @mixin 믹스인이름($매개변수A: 기본값, $매개변수B: 기본값){ 스타일; } @include 믹스인이름($매개변수B: 인수);

Scss

@mixin position( $p: absolute, $t: null, $b: null, $l: null, $r: null ){ position:$p; top:$t; bottom:$b; left:$l; right:$r; } .box1{ @include position($b:10px, $r:20px); } .box2{ @include position(fixed, $t:30px, $r:42px); }
▼

CSS

.box1{ position:absolute; bottom:10px; right:20px; } .box2{ position:fixed; top:30px; right:42px; }
